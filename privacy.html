Política de Privacidad

Última actualización: Septiembre 2025

🎯 Nuestro Enfoque de Privacidad

En Kuruba, nuestro objetivo principal es brindarte la mejor información sobre ingredientes y consumo. En la versión actual, no recopilamos datos personales porque no los necesitamos para las funcionalidades básicas de escaneo y análisis. En el futuro, podríamos implementar características que requieran información para personalizar tu experiencia y ofrecerte alertas más relevantes.

📱 ¿Qué es Kuruba?

Kuruba es una aplicación móvil que escanea ingredientes de productos en tiempo real y proporciona información relevante sobre su consumo. Te permite apuntar la cámara hacia etiquetas de productos para obtener análisis instantáneo de ingredientes y recomendaciones sobre su impacto en tu alimentación, todo procesado localmente en tu dispositivo.

📊 Información Actual vs. Futura

📊 Información que SÍ Recopilamos:

Para ofrecerte el mejor servicio y mejorar continuamente, Kuruba recopila:

  • Datos de ingredientes consultados: Registramos qué ingredientes consultas para generar métricas y mejorar nuestro servicio.
  • Patrones de uso: Analizamos cómo se utiliza la aplicación para optimizar la experiencia del usuario.
  • Datos técnicos básicos: Información necesaria para el funcionamiento de la API y resolución de problemas técnicos.
  • Imágenes para procesamiento: Las imágenes se envían a nuestra API para análisis, pero no se almacenan permanentemente.

🚫 Información que NO Recopilamos:

Mantenemos tu privacidad personal al no recopilar:

  • Información personal identificable: No recopilamos nombres, direcciones de correo electrónico, números de teléfono o datos de identificación personal.
  • Datos de ubicación: No accedemos ni recopilamos información sobre tu ubicación geográfica.
  • Contactos o archivos: No accedemos a tu lista de contactos, fotos guardadas u otros archivos personales.
  • Información financiera: No recopilamos datos de tarjetas de crédito, cuentas bancarias o información de pagos.

🔮 Funcionalidades Futuras

Estamos desarrollando características avanzadas que podrían incluir:

  • Perfiles personalizados: Para entender mejor tus preferencias alimentarias y necesidades dietéticas.
  • Alertas personalizadas: Basadas en tu historial de consumo y objetivos de salud.
  • Recomendaciones mejoradas: Adaptadas a tus patrones de alimentación y restricciones.
  • Análisis de tendencias: Para ayudarte a mantener un consumo consciente a largo plazo.
Compromiso futuro: Si implementamos estas funcionalidades, siempre te informaremos claramente qué datos recopilamos, cómo los usamos, y te daremos control total sobre tu información personal.

📸 Uso de la Cámara

Kuruba solicita acceso a la cámara de tu dispositivo únicamente para:

  • Escanear etiquetas de productos en tiempo real
  • Analizar y procesar la información de ingredientes localmente
  • Proporcionarte resultados instantáneos sin almacenamiento
Importante: Las imágenes capturadas se envían a nuestra API para análisis y procesamiento, pero no se almacenan permanentemente. Solo se utilizan para el análisis inmediato de ingredientes y luego se descartan de nuestros servidores.

🔒 Procesamiento y Análisis de Datos

Para brindarte la mejor información sobre ingredientes, Kuruba utiliza los siguientes procesos:

  • API propia: Utilizamos nuestra propia API para procesar y analizar la información de ingredientes.
  • Conexión requerida: La aplicación necesita conexión a internet para acceder a nuestra base de datos actualizada.
  • Procesamiento en la nube: El análisis de ingredientes se realiza en nuestros servidores para garantizar precisión y velocidad.
  • Métricas de uso: Recopilamos datos sobre ingredientes consultados para mejorar nuestro servicio y tomar decisiones informadas sobre futuras funcionalidades.

👥 Uso de Datos y Compartir con Terceros

Cómo utilizamos tus datos:

  • Procesar y analizar ingredientes para brindarte información precisa
  • Generar métricas agregadas para mejorar nuestro servicio
  • Tomar decisiones informadas sobre futuras funcionalidades
  • Optimizar el rendimiento y precisión de nuestra API

Compartir con terceros:

  • No vendemos tus datos personales o de uso a terceros
  • Datos agregados y anónimos pueden usarse para estudios de mercado o investigación
  • Proveedores de servicios pueden acceder a datos necesarios para mantener la infraestructura
  • Cumplimiento legal - Solo compartimos datos si es requerido por ley

🔗 Enlaces a Sitios Web Externos

Esta aplicación puede contener enlaces a sitios web externos para proporcionar información adicional sobre productos o ingredientes. No somos responsables de las prácticas de privacidad de estos sitios web. Te recomendamos revisar las políticas de privacidad de cualquier sitio web de terceros que visites.

👶 Privacidad de Menores

Kuruba está diseñada para ser segura para usuarios de todas las edades. Como no recopilamos ningún tipo de información personal, no hay riesgos de privacidad para menores de 13 años o cualquier otro grupo de edad.

📋 Permisos Requeridos

Kuruba solicita únicamente el siguiente permiso:

📸 Acceso a la Cámara

Propósito: Escanear etiquetas de productos y analizar ingredientes en tiempo real.

Uso: Solo se utiliza cuando la aplicación está activa y tú estás escaneando conscientemente un producto.

Almacenamiento: Las imágenes NO se almacenan ni se guardan en ningún lugar.

🔄 Actualizaciones de la Política

Podemos actualizar esta Política de Privacidad ocasionalmente para reflejar cambios en nuestras prácticas o por motivos operacionales, legales o regulatorios. Si realizamos cambios materiales:

  • Publicaremos la nueva Política de Privacidad en esta página
  • Actualizaremos la fecha de "Última actualización" en la parte superior
  • Te notificaremos a través de la aplicación si los cambios son significativos

Te recomendamos revisar esta política periódicamente para mantenerte informado sobre cómo protegemos tu privacidad.

🌍 Cumplimiento Legal

Esta política de privacidad cumple con las regulaciones internacionales de protección de datos, incluyendo:

  • GDPR (Reglamento General de Protección de Datos de la UE)
  • CCPA (Ley de Privacidad del Consumidor de California)
  • LGPD (Lei Geral de Proteção de Dados do Brasil)

Al no recopilar datos personales, Kuruba supera los requisitos de estas regulaciones de privacidad.
